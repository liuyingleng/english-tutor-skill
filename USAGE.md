## 快速开始

### 首次使用

在任何接入 OpenClaw 的平台（Web/微信/Telegram/Discord/QQ）对 AI 说：

```
我想学英语
```

AI 会引导你完成：
1. ✅ 学习目标设定
2. ✅ 水平评估
3. ✅ 时间安排
4. ✅ 创建飞书知识库
5. ✅ 生成学习计划
6. ✅ 开始第一课

### 日常使用

**命令式**：
```
/english start          # 开始今天的学习
/phonics learn          # 学习音标
/vocab new              # 学习新词
/vocab review           # 复习旧词
/speaking resources     # 获取口语学习资源
/progress               # 查看进度
```

**对话式**：
```
"教我音标"
"我想学新单词"
"帮我复习一下"
"推荐一些口语视频"
"我的进度怎么样？"
```

## 核心功能

### 1. 音标训练 🔤

**学习音标**
```
用户：学习音标
AI：今天学习 5 个元音音标：/iː/ /ɪ/ /e/ /æ/ /ɑː/
    
    📖 /iː/ (长元音)
    发音：嘴唇扁平，舌位高前
    例词：see, tea, meet, beat
    中文对比：类似"衣"，但更长
    
    🔊 [播放发音]
```

**音标练习**
```
用户：练习音标
AI：听音识标练习
    
    🔊 [播放] "ship"
    这个词的元音是？
    A. /iː/  B. /ɪ/  C. /e/
```

**最小对立对**
```
用户：对立对练习
AI：区分 /iː/ 和 /ɪ/
    
    🔊 sheep - ship
    🔊 beat - bit
    🔊 seat - sit
```

---

### 2. 词汇学习 📚

**学习新词**
```
用户：学新单词
AI：今天学习 20 个高频词汇
    
    1️⃣ the /ðə/
       定冠词：这个，那个
       例句：The cat is sleeping.
       
    2️⃣ be /biː/
       动词：是，存在
       例句：I am a student.
       
    [继续...]
```

**智能复习（SRS）**
```
用户：复习单词
AI：你有 15 个单词需要复习
    
    📝 have /hæv/
    你还记得这个词的意思吗？
    
    [用户回答]
    
    ✅ 正确！下次复习：3天后
    或
    ❌ 再看一遍：有，拥有
       下次复习：明天
```

**词汇测验**
```
用户：测试我的词汇
AI：词汇测验（20题）
    
    1. "book" 的意思是？
       A. 书  B. 看  C. 预订  D. A和C
    
    2. 选择正确的句子：
       A. I have a book.
       B. I has a book.
```

---

### 3. 语法学习 📖

**今日语法**
```
用户：今天学什么语法
AI：今日语法：现在进行时
    
    📌 结构：be + 动词ing
    
    用法：表示正在进行的动作
    
    例句：
    - I am studying English.
    - She is reading a book.
    - They are playing football.
    
    练习题：
    1. He ___ (watch) TV now.
    2. We ___ (learn) English.
```

### 5. 口语练习 🗣️

**场景对话**
```
用户：练习口语
AI：选择场景：
    1. 面试
    2. 会议
    3. 餐厅
    4. 机场
    5. 日常聊天
    
用户：3
AI：餐厅场景对话
    
    服务员：Good evening! Welcome to our restaurant.
    你：[你的回答]
    
    服务员：Would you like to see the menu?
    你：[你的回答]
```

**发音纠正**
```
用户：[语音输入] "I want to improve my pronunciation"
AI：发音分析：
    ✅ "I want to" - 很好
    ⚠️ "improve" - /ɪmˈpruːv/ 注意重音在第二音节
    ✅ "my pronunciation" - 正确
    
    建议：多练习 "improve" 的发音
```

## 使用场景

### 场景1：零基础入门

**目标**：从零开始学英语

**学习路径**：
1. 第1-2周：音标学习
2. 第3-4周：高频词汇100个
3. 第5-8周：基础语法
4. 第9周+：听说读写综合训练

**每日安排**（2小时）：
- 音标练习：30分钟
- 词汇学习：40分钟
- 语法学习：30分钟
- 听力训练：20分钟

---

### 场景2：技术文档阅读

**目标**：能流畅阅读英文技术文档

**学习路径**：
1. 技术词汇专项（500个）
2. 长难句分析
3. 技术文章精读
4. 文档快速浏览技巧

**每日安排**（1小时）：
- 技术词汇：20分钟
- 文章精读：30分钟
- 快速阅读：10分钟

---

### 场景3：口语提升

**目标**：能流利进行日常和职场交流

**学习路径**：
1. 发音纠正（音标+连读）
2. 常用口语表达
3. 场景对话练习
4. 影子跟读训练

**每日安排**（1.5小时）：
- 发音练习：20分钟
- 口语表达：30分钟
- 对话练习：20分钟
- 影子跟读：20分钟

**推荐资源**：
```
/speaking resources
```

---

### 场景4：考试备考

**目标**：通过雅思/托福/四六级

**学习路径**：
1. 词汇突击（考试词汇）
2. 语法强化
3. 听力专项
4. 写作模板
5. 真题练习

**每日安排**（3小时）：
- 词汇：1小时

## 常见问题

### Q1: 如何切换学习平台？

**A**: 所有数据存储在飞书，可以在任何平台无缝切换：
- 早上在电脑上学习（Web）
- 通勤路上用手机（微信/Telegram）
- 晚上在平板上复习（Discord）

### Q2: 如何调整学习计划？

**A**: 对 AI 说：
```
我想调整学习计划
```

AI 会引导你重新设置目标和时间。

### Q3: 忘记复习怎么办？

**A**: AI 会主动提醒：
```
🔔 你有 15 个单词需要复习，现在复习吗？
```

也可以手动查看：
```
/vocab review
```

### Q4: 如何导出学习数据？

**A**: 所有数据在飞书知识库，可以：
1. 在飞书中查看和导出
2. 使用命令：`/export data`

### Q5: 支持离线学习吗？

**A**: 部分支持：
- ✅ 本地词库可离线使用
- ✅ 已下载的材料可离线查看
- ❌ 飞书同步需要网络
- ❌ AI 对话需要网络

### Q6: 如何获取更多学习资源？

**A**: 使用资源搜索功能：
```
推荐口语学习视频
推荐英语播客
推荐技术英语文章
```

AI 会根据你的水平和兴趣推荐资源。

### Q7: 学习进度会丢失吗？

**A**: 不会，所有数据实时同步到飞书：
- 词汇库
- 学习记录
- 进度统计
- 学习计划

### Q8: 可以多人使用吗？

**A**: 当前版本是个人使用，每个人需要：
- 自己的 OpenClaw 账号
- 自己的飞书空间

未来可能支持多用户和学习小组。

### Q9: 如何反馈问题？

**A**: 
- GitHub Issues: https://github.com/liuyingleng/english-tutor-skill/issues
- 直接对 AI 说："报告问题"

### Q10: 如何贡献内容？

**A**: 欢迎贡献：
- 词汇库扩充
- 学习材料
- 练习题
- 功能建议

参考：[贡献指南](CONTRIBUTING.md)
